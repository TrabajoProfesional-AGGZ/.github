# 📦 Repositorios

El proyecto **SocioUnido** está construido sobre una arquitectura distribuida. A continuación se detallan los repositorios principales que conforman nuestro ecosistema técnico:

* [**Plataforma Web (Frontend)**](#)
  * Interfaz de usuario principal y aplicación web progresiva (PWA) para administradores y socios.
* [**API Gateway**](#)
  * Puerta de enlace centralizada encargada del enrutamiento seguro de todas las peticiones.
* [**Microservicio de Autenticación (Auth)**](#)
  * Gestión de identidades, inicio de sesión mediante JWT, control de roles y permisos de usuarios.
* [**Microservicio Club (Core)**](#)
  * Lógica central del negocio: ABM de socios, gestión de disciplinas y control de instalaciones.
* [**Healthchecker & CI/CD**](#)
  * Herramientas de despliegue continuo, validación de código, flujos de GitHub Actions y monitoreo de estado.
* [**Documentación Oficial**](#)
  * Repositorio central de la documentación técnica, ceremonias Scrum y bitácoras (este mismo sitio).

*(Nota: Modificar los enlaces "#" con las URLs exactas de cada repositorio en GitHub)*
